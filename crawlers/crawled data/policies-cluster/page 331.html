inCarDoc Privacy Policy

General information

inCarDoc is an application that allows user to analyze the status of various parameters of the engine and cars in real time using a smartphone for further analysis and driving optimizing.

Features

  • Real-time engine and vehicle OBD2 parameters
  • Chart drawing for real-time data
  • Check engine light control (MIL)
  • Fuel Economy/Consumption parameters (MPG)
  • Code scanner
  • Auto-start of last used command screen
  • Console for advanced users experiments with OBD-II and adapter commands
  • Read GPS data: speed, altitude
  • Support of graphical gauges
  • Logbook: fueling, service records, car maintenance, diagnostics

Information Collected by inCarDoc

We may collect data and information to further improvement of our products and services.

  • Vehicle year, make, and model
  • Anonymized VIN (stripped of the vehicle serial number)
  • Supported PIDs, OBD2 protocol, and OBD standard
  • Application name and version
  • OBD2 communications, which includes information about your scan tool and vehicle
  • Region
  • Statistics of usage
  • Any error messages that may have occurred while you were using the application
  • Debug Log

General data is collected by Google Analytics and will not be shared with third parties.
