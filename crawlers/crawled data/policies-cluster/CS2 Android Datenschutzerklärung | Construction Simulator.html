CS2 Android Datenschutzerklärung

Datenschutzerklärung der Firma astragon für mobile Applikationen

Vielen Dank für Ihr Interesse an unseren Produkten und herzlich willkommen auf unserer Seite zur Datenschutzerklärung! Wir sind die Firma astragon Entertainment GmbH, Am Wehrhahn 33, D-40211 Düsseldorf, nachfolgend astragon genannt.

1. Worum geht es?

Im Folgenden möchten wir Ihnen kurz darlegen, wie astragon zum Datenschutz steht, wie wir Ihre Daten schützen und was es für Sie bedeutet, wenn Sie unsere Programme nutzen. Der Schutz Ihrer Privatsphäre ist für uns von höchster Bedeutung. Deshalb ist das Einhalten der gesetzlichen Bestimmungen zum Datenschutz für uns selbstverständliche Grundlage. Darüber hinaus ist es uns aber wichtig, dass Sie jederzeit wissen, wann wir welche Daten erheben, speichern und wie wir sie verarbeiten. Jederzeit können Sie diese Erklärung unter der URL http://www.bau-simulator.de/de/meta/cs2-android-data-privacy abrufen. Nehmen Sie diese Datenschutzerklärung an, wird sie Bestandteil der EULA. Damit alle von den gleichen Dingen reden, definieren wir hier ein paar Begriffe. astragon richtet sich dabei nach dem deutschen Bundesdatenschutzgesetz. Personenbezogene Daten sind Einzelangaben über persönliche oder sachliche Verhältnisse einer bestimmten oder bestimmbaren natürlichen Person. Das heißt im „Nicht-Juristen-Deutsch“: Personenbezogen sind alle Daten, durch die Sie persönlich identifizierbar sind, wie z.B. Adresse, Name, Telefonnummern usw. Alle Daten, mit deren Hilfe Ihre Identität NICHT festgestellt werden kann, sind KEINE personenbezogenen Daten! Erheben von Daten ist das Beschaffen von Daten über den Betroffenen. Speichern von Daten ist das Erfassen, Aufnehmen und Aufbewahren personenbezogener Daten auf einem Datenträger zum Zweck der weiteren Verarbeitung oder Nutzung. Verarbeiten von Daten ist das Speichern, Verändern, Übermitteln, Sperren und Löschen personenbezogener Daten.

2. Wann, wie und wozu werden die Daten erhoben?

astragon ist nur an zusammengefassten Informationen interessiert, die uns helfen, die Qualität unsere Spiele immer weiter zu verbessern und den Kundenwünschen noch mehr anzupassen. Hierfür werden von uns Dienste von Drittanbietern genutzt. Welche das bei Ihrer App sind, steht unter Punkt 3. In dem Fall, dass Sie eine Softwareversion nutzen, bei der Werbung für Produkte von Fremdfirmen eingeblendet wird, nutzen wir andere Dienste von Drittanbietern. Welche das bei Ihrer App sind, steht unter Punkt 4. Diese Drittanbieter blenden (nur bei diesen Softwareversionen!) in das von Ihnen genutzte astragon-Produkt Werbung ihrer eigenen Kunden ein. Auf die dazu erforderlichen personenbezogenen Daten haben wir keinen Einfluss. Damit Sie unsere App nutzen können, müssen Sie sie zuerst downloaden. Im Rahmen des Downloads werden von der jeweiligen Plattform, also Apple ITunes, Google Play Store oder dem Microsoft Store Daten erhoben, gespeichert und verarbeitet. Welche Plattform auf Sie zutrifft, steht in Punkt 6. Auf diese Datenerhebung haben wir keinen Einfluss.

3. Wie, durch wen und wann werden Daten zur Qualitätssicherung erhoben?

Wie schon geschrieben ist astragon nur an zusammengefassten Informationen interessiert die uns helfen, die Qualität unsere Spiele immer weiter zu verbessern und den Kundenwünschen noch mehr anzupassen. Hierfür werden von uns Dienste von Drittanbietern genutzt. Die dabei gesammelten Daten stehen astragon ausschließlich in anonymisierter Form zur Verfügung! Sie werden für die Dauer des Vertriebs des Spiels gespeichert und können auch so lange verarbeitet werden. astragon setzt zu diesem Zweck eine Software namens GameAnalytics ein. Die Software GameAnalytics wurde von GameAnalytics ApS, Vesterbrogade 34, 4th floor, DK-1620 Copenhagen V, Denmark (nachfolgend als „GameAnalytics“ bezeichnet) entwickelt. Die Datenschutzbestimmungen von GameAnalytics können Sie unter diesem Link einsehen: http://www.gameanalytics.com/privacy GameAnalytics verwendet Cookies, die auf Ihrem Endgerät gespeichert werden und eine Analyse Ihres Benutzerverhaltens ermöglichen. Die durch den Cookie erzeugten Informationen werden von GameAnalytics auch im Ausland, insbesondere auf Servern in Dänemark und den Vereinigten Staaten von Amerika, gespeichert. Die Speicherung der durch Game Analytics eingesetzten Cookies können Sie ausschließen, indem Sie im Bereich „Einstellungen“ Ihres Endgeräts die Verwendung von Cookies ausschließen. In diesem Fall ist die Funktionsweise Ihrer App jedoch unter Umständen stark eingeschränkt. Astragon setzt zu Analysezwecken außerdem die Software „FireBase“ ein. Die Software „FireBase“ wurde von Google, 1600 Amphitheatre Parkway, Mountain View, CA 94043, USA (nachfolgend als „Google bezeichnet) entwickelt.

Nutzerdaten werden ausschließlich anonymisiert an Firebase übermittelt. Die Datenschutzerklärung von Google für die Software „Firebase“ finden Sie unter nachstehendem Link:

https://www.firebase.com/terms/privacy-policy.html.

4. Wer erhebt die Daten bei werbefinanzierten Produkten?

Ihr Produkt ist nicht werbefinanziert.

5. Gibt es eine interaktive Anbindung an Facebook?

Sie können sich im Rahmen des Spiels mit Ihrem Facebook-Profil verbinden. Auf diesem Wege ist es möglich Erfolge und Screenshots mit Ihrem Facebook Netzwerk zu teilen und an Marketingaktionen teilzunehmen (z.B. Belohnung für einen Facebook Like). Aufgrund dieser Verbindung werden die folgenden von Facebook erhobenen Daten astragon zur Verfügung gestellt: E-Mail Adresse; das öffentliche Profil, (sofern bei Facebook hinterlegt, der Vorname, der Nachname; das Profilfoto, Geschlecht sowie das Alter); Freundesliste die ebenfalls die Spiele-App nutzen. Astragon gibt die folgenden Daten an Facebook weiter: Name der Marketingaktionen Durch die Verknüpfung mit Ihrem Facebookprofil erklären Sie sich mit der Datenverarbeitung sowie -weitergabe zu den genannten Zwecken einverstanden. Astragon hat auf die Datenerhebung von Facebook keinen Einfluss. Wir verweisen auf folgende Datenschutzrichtlinie: https://www.facebook.com/about/privacy.

6. Werden durch den Anbieter der Download Plattform Daten erhoben?

Je nachdem, welche Hardware Sie haben (Apple Produkt, Android Gerät oder ein Windows Phone Produkt), wird Ihnen die App über die entsprechend passende Plattform angeboten. Von dem Anbieter der Plattform werden Daten erhoben. astragon hat auf diese Daten keinen Einfluss und auch keinen Zugriff auf die Daten. Im Fall der Verwendung eines Android Geräts werden im Rahmen des Downloads Daten durch Google, 1600 Amphitheatre Parkway, Mountain View, CA 94043, USA, erhoben, gespeichert und verarbeitet. Wir verweisen auf folgende Datenschutzrichtlinie https://play.google.com/intl/de_de/about/play-terms.html.

Die hier erteilte Einwilligung erfolgt freiwillig und kann mit Wirkung für die Zukunft widerrufen werden, soweit die Daten im Rahmen der Qualitätssicherung (siehe Punkt 3.) durch astragon erhoben wurden.
