Privacy Policy

Your Consent.
By using this website, you consent to this website’s (www.retro-8bit.com; “Retro8 Website”) privacy policy.

What is each Retro8 application permission used for?

Full network access - Downloading covert art, video effects and game cheats. Logging Google analytics activity.
Google Play license check - May choose to check the validity of a license.
View Wi-Fi connections - Used to check if Wi-Fi is available for multiplayer.
View network connections - Used to check if local network is available for multiplayer.

Allow Wi-Fi Multicast reception - Briefly used to broadcast available multiplayer games on local network
Control vibration - Touch screen vibration. Optionally may be turned off.

Pair with Bluetooth devices - Used for pairing with other devices in multiplayer.
Access Bluetooth settings - Used to check if Bluetooth is enabled and detect external controllers.

Modify or delete the contents of your USB storage - Used to write game save state files and delete NES game files at user's request.
Test access to protected storage - Used to scan storage for NES game files.

Identity - Used solely to store purchase information across user's devices.
In-app Purchases - Used to allow users to buy products in the emulator.

Do we use cookies?

Neutron Emulation uses Google Analytics, which utilizes cookies, to help analyze how users use the Retro8 Website. Google Analytics only collects data obtained when such users access the Retro8 Website with a standard web browser. The Google Analytics’ cookies then generate information that is transmitted to Google about your use of the Retro8 Website, including visited IP addresses. This information is then used by Neutron Emulation to compile statistical reports on Retro8 Website activity and evaluate users use of the Retro8 Website.  Neutron Emulation will never (and will not allow any third party to) use this Analytics tool to track or to collect any of your personally identifiable information pertaining to the Retro8 Website. Google will not associate your IP address with any other data held by Google. Neither Google nor we will link, or seek to link, an IP address with the identity of a computer user. Neutron Emulation will not associate any data gathered from the Retro8 Website with any personally identifiable information from any source, unless Retro8 Website users explicitly submit that information via a fill-in form from the Retro8 Website. Further information on Google Analytics’ own privacy policies can be found in Google's Privacy Policy.
