Permissions required by the App

  • WRITE_EXTERNAL_STORAGE: read and write game save files on internal/external storage, so you will be able to store/restore the emulator status, memcards and game configuration profiles.
  • INTERNET: to download cheat codes used in the games.
  • VIBRATE: the ps1 supported vibration during gameplay, it is emulated using vibration in the phone.
  • CHECK_LICENSE: to check the app licence on Google play.
  • INSTALL_SHORTCUT: to install shortcuts to the games in the mobile desktop.
  • BLUETOOTH/BLUETOOTH_ADMIN: used to support bluetooth gamepads.
